# Sample Packs

Voeg je eigen sample packs toe in deze map.

## Mapstructuur

```
samples/
  mijn-pack/
    kick/
      kick1.wav
      kick2.wav
      kick_808.wav
    snare/
      snare1.wav
      clap.wav
    hihat/
      hihat_closed.wav
    open_hat/
      hihat_open.wav
    clap/
      clap1.wav
    perc/
      shaker.wav
      conga.wav
    tom/
      tom_low.wav
      tom_mid.wav
    cymbal/
      crash.wav
      ride.wav
    fx/
      riser.wav
      downlifter.wav
```

## Ondersteunde formaten

WAV, MP3, OGG, AIFF, FLAC

## Gebruik

1. Maak een map aan met de naam van je pack (bijv. `samples/trap-kit/kick/`)
2. Voeg audiobestanden toe in de juiste categoriemap
3. Open ShotMusic → klik het `◈ Sample` knopje op een drumtrack
4. Kies je pack en sample in de browser

## Tips

- Gebruik korte, bondige bestandsnamen (worden getoond in de UI)
- WAV of AIFF geeft de beste kwaliteit en laadsnelheid
- MP3 is kleiner maar heeft lichte vertraging bij het laden
